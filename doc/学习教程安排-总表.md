# 完整 8 周本体建模学习教程安排（总表）

> ✅ = 必做  
> 🧠 = 思考题  
> 🛠 = 实操任务

|---|---|---|---|---|

## 给这份计划的 3 条使用建议

1. **每周只做“必做任务”即可**，不要贪多；多出来的精力用来改上周的本体。
2. **第4周是最关键的转折点**：从“照着做”变成“自己设计”。
3. 第8周结束后，你已经有能力：
   - 看懂领域本体
   - 独立设计一个中型本体
   - 用本体约束大模型输出
