# 第1周实战：用 Protégé 构建「University 大学本体」
**目标**：学会建类、建属性、建实例，并用推理机验证结构。

---

## 一、准备工作（5分钟）

### 1. 下载安装 Protégé
1. 打开官网：https://protege.stanford.edu/
2. 点击 **Download Protégé**
3. 选择你的系统（Windows / macOS / Linux）
4. 解压即可，**不需要安装**，双击 `Protege.exe`（Windows）或 `Protege.app`（macOS）启动

> 第一次打开可能会慢一点，耐心等待。

### 2. 新建本体
1. 打开 Protégé
2. 菜单栏 → **File → New**
3. 右侧 **Active Ontology** 标签页
4. 找到 **Ontology IRI**，改成：
   ```
   http://www.example.org/university#
   ```
5. 点击 **OK**

✅ 你现在已经有了一个空的本体。

---

## 二、建类（Classes）（10分钟）

### 1. 进入 Classes 标签页
左侧点击 **Entities → Classes**

### 2. 创建顶层类 `Person`
1. 选中 `owl:Thing`
2. 点击工具栏上的 **Add subclass (⊕)**
3. 输入类名：`Person`
4. 回车

### 3. 创建子类 `Student` 和 `Teacher`
1. 选中 `Person`
2. 点击 **Add subclass**
3. 输入：`Student`
4. 再点一次 **Add subclass**
5. 输入：`Teacher`

现在你的类层级应该是：
```
owl:Thing
 └── Person
      ├── Student
      └── Teacher
```

📌 **小提示**：
- 类名首字母大写是约定俗成
- 这里的关系是 **is-a**：Student **is a** Person

---

## 三、建对象属性（Object Properties）（10分钟）

### 1. 进入 Object Properties 标签页
左侧切换到 **Object Properties**

### 2. 创建属性 `teaches`
1. 选中 `owl:topObjectProperty`
2. 点击 **Add subproperty**
3. 输入：`teaches`

### 3. 设置定义域（Domain）
1. 选中 `teaches`
2. 右侧找到 **Domain (intersection)** → 点击 **+**
3. 选择 `Teacher` → OK

👉 含义：**只有老师才能“教”**

### 4. 设置值域（Range）
1. 右侧找到 **Range (union)** → 点击 **+**
2. 选择 `Student` → OK

👉 含义：**教的一定是学生**

---

## 四、建数据属性（Data Properties）（5分钟）

### 1. 切换到 Data Properties
左侧点击 **Data Properties**

### 2. 创建 `hasName`
1. 选中 `owl:topDataProperty`
2. 点击 **Add subproperty**
3. 输入：`hasName`
4. 右侧 **Range** → 点击 **+**
5. 选择 `xsd:string` → OK

### 3. 创建 `hasAge`
1. 同样方式新建 `hasAge`
2. Range 选择 `xsd:int`

---

## 五、建实例（Individuals）（10分钟）

### 1. 切换到 Individuals 标签页
左侧点击 **Individuals by class**

### 2. 创建老师 `Alice`
1. 展开 `Teacher`
2. 右键 `Teacher` → **Add individual**
3. 名称输入：`Alice`
4. 右侧 **Data property assertions** → 点击 **+**
   - 属性：`hasName`
   - 值：`"Alice"`
5. 再点 **+**
   - 属性：`hasAge`
   - 值：`35`

### 3. 创建学生 `Bob`
1. 展开 `Student`
2. 右键 `Student` → **Add individual**
3. 名称：`Bob`
4. 添加数据属性：
   - `hasName` → `"Bob"`
   - `hasAge` → `20`

### 4. 建立“教”的关系
1. 选中 `Alice`
2. 右侧 **Object property assertions** → 点击 **+**
3. 属性：`teaches`
4. 目标：`Bob` → OK

✅ 现在你已经表达了：  
**Alice（老师，35岁）教 Bob（学生，20岁）**

---

## 六、保存你的成果（2分钟）

1. 菜单栏 → **File → Save**
2. 文件名：`university-v1.owl`
3. 保存到桌面

🎉 **恭喜！你的第一个本体已经诞生了。**

---

## 七、检查你是否真的学会了（自检清单）

你可以对照一下，看看这些你都能解释清楚吗：

- [ ] `Student` 和 `Person` 是什么关系？
- [ ] `teaches` 的 Domain 和 Range 分别限制了什么？
- [ ] 如果把 `Bob` 的类型改成 `Teacher`，逻辑上会发生什么？
- [ ] 数据属性和对象属性的本质区别是什么？

---

## 八、下周预告（第2周你要做的事）

下周我们会在这个本体上加**约束**，比如：
- 一门课必须有老师
- 一个学生最多选 5 门课
- 老师不能教自己

这会让你第一次感受到：**本体不是“画框图”，而是“写规则”**。
